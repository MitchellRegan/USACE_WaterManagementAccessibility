Info

  • This web application was developed for the USACE by Mitchell Regan and Matthew Postel.
  • This page is the admin-only page for adding box-annotations to the graphs.
  • Press the ⓘ in the corner to bring this up or close this at any time!
  • Upon loading the page on a mobile device, a prompt should appear to "Add Water Data to Home screen" (at least on Android). This is entirely optional, but is for your convenience! Simply tap the prompt and hit install!
  • Speaking of mobile, you will mostly want to use the application in landscape mode for best use.

Steps of use:

  1. Enter the name of the event into the "Event Title" box.
  2. Also write out the description into the "Event Description" box.
  3. If you have one, add a direct link to an image you desire displayed with the info.
  4. Enter the start date and time of the event.
  5. Enter the end date and time of the event.
  6. Select the timezone the event took place in.
  7. Use the search bar in the right pane to find a water location the event affected. The results are sorted in no particular order.
  8. Select a location from the list of options. Selecting one will fill in the name into the search bar.
  9. Press "Add To List".
  10. The name of the location will be added to a list below the search bar.
    • The "X" button may be pressed to remove the location from the list.
  11. Continue to add each affected site.
  12. When all locations have been added, press the "Submit" button to send the event to the database.
    • If you are not signed in with a whitelisted account, your event will be rejected.
    • If there are any errors, the system will prompt you further with advice to fix it. Be sure to read carefully.
